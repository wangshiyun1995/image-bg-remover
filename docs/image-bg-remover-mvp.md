# Image Background Remover - MVP 需求文档

## 1. 项目概述

### 1.1 产品名称
Image Background Remover（图片背景移除工具）

### 1.2 产品定位
一个简洁、快速的在线图片背景移除工具，用户上传图片后即可获得透明背景的处理结果。

### 1.3 目标用户
- 电商卖家（需要商品白底图/透明图）
- 设计师（快速抠图辅助）
- 普通用户（证件照换背景、创意图片制作）

---

## 2. 核心功能（MVP 范围）

### 2.1 图片上传
- 支持拖拽上传
- 支持点击选择文件上传
- 支持格式：JPG、PNG、WEBP
- 单文件大小限制：5MB
- 仅支持单张图片处理（MVP 阶段）

### 2.2 背景移除处理
- 调用 remove.bg API 进行背景移除
- 返回透明背景 PNG 图片
- 处理时间目标：< 5 秒

### 2.3 结果展示与下载
- 处理前后对比展示（左右对比或切换对比）
- 提供下载按钮，保存处理后的图片
- 不存储任何图片数据（纯内存处理）

---

## 3. 技术架构

### 3.1 技术栈
| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Next.js 14+ (App Router) | React + TypeScript |
| 样式 | Tailwind CSS | 原子化 CSS |
| UI 组件 | shadcn/ui 或自定义 | 简洁风格 |
| 部署平台 | Cloudflare Pages | 边缘部署 |
| 图片处理 | remove.bg API | 第三方背景移除服务 |

### 3.2 架构图
```
用户浏览器
    ↓ 上传图片 (Base64/FormData)
Cloudflare Edge (Next.js)
    ↓ 调用 API
remove.bg API
    ↓ 返回透明图
Cloudflare Edge
    ↓ 返回给用户
用户浏览器 (预览 + 下载)
```

### 3.3 数据流
1. 用户在前端选择/拖拽图片
2. 前端将图片转为 Base64 或 FormData
3. 调用 Next.js API Route (`/api/remove-bg`)
4. API Route 调用 remove.bg API
5. 获取处理结果后直接返回给前端
6. 前端展示对比图，提供下载

## 5. API 设计

### 5.1 内部 API

#### POST /api/remove-bg
移除图片背景

**请求**
```json
{
  "image": "base64encodedstring..."
}
```

**响应**
```json
{
  "success": true,
  "image": "base64encodedresult...",
  "format": "png"
}
```

**错误响应**
```json
{
  "success": false,
  "error": "文件过大，请上传小于 5MB 的图片"
}
```

### 5.2 外部 API 集成

#### remove.bg API
- 端点：`POST https://api.remove.bg/v1.0/removebg`
- 认证：API Key (Header: `X-Api-Key`)
- 请求体：FormData (`image_file` 或 `image_url`)
- 响应：二进制图片数据（透明背景 PNG）

## 6. 非功能需求

### 6.1 性能
- 页面首屏加载时间 < 2 秒
- 图片处理总耗时 < 10 秒（含网络传输）
- API 超时设置：30 秒

### 6.2 安全
- 不存储任何用户图片数据
- 前端验证文件类型和大小
- API Key 仅存储在服务端环境变量
- 限制单 IP 请求频率（可选）

### 6.3 兼容性
- 支持现代浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，支持移动端
- 渐进增强，基础功能在无 JS 时仍可提示

### 6.4 可访问性
- 图片上传区域支持键盘操作
- 按钮和交互元素有明确的焦点状态
- 处理状态有文字和视觉反馈
